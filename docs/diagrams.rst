Диаграммы
=========

Исходники диаграмм в формате PlantUML находятся в директории ``diagrams/``.

Для рендера: https://www.plantuml.com/plantuml/uml/ или VS Code PlantUML extension.

Контекстная диаграмма (IDEF0)
------------------------------

.. code-block:: none

   diagrams/context.puml

Описывает внешних акторов системы: Турист, Менеджер, Горничная, Администратор
и внешние системы: OpenStreetMap, Эквайринг, Лавка.

Диаграмма последовательности
-----------------------------

.. code-block:: none

   diagrams/sequence.puml

Процесс бронирования номера от запроса пользователя до подтверждения.

ERD (схема данных)
-------------------

.. code-block:: none

   diagrams/erd.puml

Все 11 таблиц с первичными и внешними ключами.
